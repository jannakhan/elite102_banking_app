# elite102_banking_app
This is a simple banking app I built using Python and Flask.
You can create an account, deposit money, withdraw money, and check your balance.

How to run:
- Make sure you have Python installed
- Install Flask:
    - python -m pip install flask
- Go into the project folder:
    - cd elite102_banking_app
- Run the app:
    - python app.py
- Open in browser:
    - http://127.0.0.1:5000
